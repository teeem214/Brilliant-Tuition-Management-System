# Brilliant Tuition Centre (BTC) Management System
A Python-based management console designed to automate operations for a local tuition business.

# Project Overview
This application serves as a centralized system for managing tutors, students, and administrative tasks. It provides a structured way to handle schedules, subject assignments, and student enrollments.

# Key Features
- Tutor Management: Functions to add, view, and assign tutors to specific academic subjects.
- Student Enrollment: Automates the process of registering students and linking them to appropriate classes.
- Administrative Operations: Streamlines business workflows for a small-scale educational enterprise.
- Logic-Driven Design: Utilizes Python control structures and data types to ensure efficient data handling.

# Tech Stack
- Language: Python
- Development Environment: PyCharm
